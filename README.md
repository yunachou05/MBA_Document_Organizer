# MBA_Document_Organizer
MBA_Document_Organizer is a Python tool that automates organizing and analyzing academic documents. It scans local folders, extracts key information, and generates a structured Excel report for easy tracking. This project was developed with the assistance of AI (ChatGPT) as a coding partner for problem-solving, debugging, and feature enhancement.
## Features
- Scan folders and subfolders automatically
- Extract text from PDF and Word files
- Identify professor names and emails
- Generate Excel report
- Show progress bar and percentage

## Technologies
- Python
- pandas
- pdfplumber
- python-docx
- tqdm

## Output
The tool generates an Excel file with:
- File Name
- Folder Path
- File Type
- Professor
- Email

## Collaboration
Developed with the assistance of AI (ChatGPT)

## How to Run
pip install -r requirements.txt
python app.py
